## FÓRMULAS CRÍTICAS (MEMORIZAR)

### Risk Management
```
SLE (Single Loss Expectancy) = Asset Value × Exposure Factor
ALE (Annual Loss Expectancy) = SLE × ARO
ARO (Annual Rate of Occurrence) = veces/año

ROI = (ALE_before - ALE_after - Control_Cost) / Control_Cost × 100%
```

### Subnetting
```
Hosts disponibles = 2^(32 - CIDR) - 2

/24 = 255.255.255.0     = 254 hosts
/25 = 255.255.255.128   = 126 hosts
/26 = 255.255.255.192   = 62 hosts
/27 = 255.255.255.224   = 30 hosts
/28 = 255.255.255.240   = 14 hosts
```

### Business Continuity
```
MTBF (Mean Time Between Failures) = Uptime / Número de fallos
MTTR (Mean Time To Repair) = Downtime total / Número de reparaciones
Availability = (Total_Time - Downtime) / Total_Time × 100%

RTO (Recovery Time Objective) = Tiempo máximo aceptable de downtime
RPO (Recovery Point Objective) = Pérdida máxima aceptable de datos
```

### CVSS (Common Vulnerability Scoring System)
```
0.0           = None
0.1-3.9       = Low
4.0-6.9       = Medium
7.0-8.9       = High
9.0-10.0      = Critical
```
