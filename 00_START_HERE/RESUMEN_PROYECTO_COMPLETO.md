## 🔑 CONCEPTOS CASI SEGUROS EN EXAMEN

### Memoriza estos (CASI SEGURO aparecen):

**1. Orden Respuesta a Incidentes (IR)**
```
1. Preparation
2. Identification
3. Containment
4. Eradication
5. Recovery
6. Lessons Learned
```

**2. Orden de Volatilidad (Forense)**
```
1. CPU, caché
2. Memoria RAM
3. Estado de red
4. Procesos
5. Archivos temporales
6. Disco duro
7. Logs remotos
8. Backups
```

**3. Fórmulas Risk Management**
```
SLE = Asset Value × Exposure Factor
ALE = SLE × ARO
ROI = (ALE_before - ALE_after - Cost) / Cost × 100%
```

**4. Subnetting**
```
Hosts = 2^(32 - CIDR) - 2

/24 = 254 hosts
/26 = 62 hosts
/28 = 14 hosts
/30 = 2 hosts
```

**5. Windows Event IDs**
```
4624 = Logon exitoso
4625 = Logon fallido
4672 = Privilegios especiales
4688 = Proceso creado
4720 = Usuario creado
4732 = Usuario añadido a grupo
```

**6. CIA Triad**
```
Confidentiality = Encryption, Access Control, MFA
Integrity = Hashing, Digital Signatures, Checksums
Availability = Backups, RAID, Redundancy, UPS
```
